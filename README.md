# 🪨 Rock, Paper, Scissors Game 📄 ✂️

A classic Rock, Paper, Scissors game built with HTML, CSS, and JavaScript. Challenge the computer in this interactive web-based game where the first to score 3 points wins!

## ✨ Features

- **Interactive Gameplay**: Play against the computer with real-time results
- **Score Tracking**: Keeps track of player and computer scores
- **First to 3 Wins**: The game ends when either player reaches 3 points
- **Game Reset**: Option to play again after the game ends
- **Rules Section**: Expandable rules section explaining game mechanics
- **Responsive Design**: Works on desktop and mobile devices

## 🎯 How to Play

1. Click on **Rock**, **Paper**, or **Scissors** to make your choice
2. The computer randomly selects its option
3. The winner of each round gets 1 point
4. First player to reach 3 points wins the game
5. Click "Play again?" to start a new match

## 📋 Game Rules

- 🪨 **Rock beats Scissors**
- ✂️ **Scissors beats Paper**
- 📄 **Paper beats Rock**
- Same choices result in a tie (no points awarded)
